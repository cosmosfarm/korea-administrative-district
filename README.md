## korea-administrative-district

대한민국의 행적 구역 데이터를 JSON 형식으로 정리했습니다.

대한민국의 행정 구역(大韓民國의 行政 區域)은 대한민국의 통치권을 행사하는 지역에서 1개의 특별시, 6개의 광역시, 8개의 도, 1개의 특별자치시, 1개의 특별자치도로 구성된다. 이상 총 17개의 행정구역은 광역지방자치단체로 분류된다.
